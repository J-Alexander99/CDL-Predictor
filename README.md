# CDL Match Predictor

A modern Python application for scraping Call of Duty League data and predicting match outcomes using machine learning.

## Features

- Web scraping of CDL match data and player statistics
- Team performance tracking and analysis
- Match outcome prediction using historical data
- Clean, maintainable codebase with proper error handling

## Project Structure

```
CDL-Predictor/
├── src/
│   ├── scrapers/          # Web scraping modules
│   ├── models/            # Data models (Team, Player, Match)
│   ├── data/              # Data processing and storage
│   ├── predictor/         # ML prediction logic
│   └── utils/             # Utility functions
├── config/                # Configuration files
├── data/                  # Stored data (gitignored)
├── tests/                 # Unit tests
├── requirements.txt       # Python dependencies
└── main.py               # CLI entry point
```

## Setup

```bash
# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# Install dependencies
pip install -r requirements.txt
```
